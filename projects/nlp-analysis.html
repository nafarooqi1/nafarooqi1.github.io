Text Mining & NLP Analysis

Applied NLTK and TF-IDF on unstructured text data for sentiment analysis and topic modeling, translating raw text into structured analytical findings.

📝 NLP — 2025PythonNLTKTF-IDFSentiment Analysis

Overview

This project applies core NLP techniques to unstructured text corpora, building pipelines that clean, tokenize, vectorize, and model text data. The goal was to extract meaningful patterns — sentiment trends and latent topics — that are not visible in raw text.

Techniques Applied

  • Text Preprocessing: Lowercasing, punctuation removal, stopword filtering, stemming/lemmatization
  • Tokenization: Word and sentence-level tokenization with NLTK
  • TF-IDF Vectorization: Converting text to numerical features capturing term importance
  • Sentiment Analysis: Polarity scoring using VADER and lexicon-based approaches
  • Topic Modeling: LDA for latent topic discovery across document corpus
