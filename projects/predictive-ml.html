Project Overview

This project demonstrates end-to-end machine learning proficiency by building, comparing, and evaluating multiple model types on real datasets. Each pipeline covers the full workflow: data loading and cleaning, exploratory analysis, feature engineering, model training, evaluation, and interpretation of results.

Models & Techniques Covered

  • Supervised — Regression: Linear Regression, Ridge, Lasso; RMSE and R² evaluation
  • Supervised — Classification: Logistic Regression, Decision Tree, Random Forest, KNN; accuracy, precision, recall, F1
  • Unsupervised — Clustering: K-Means, DBSCAN; elbow method, silhouette scores
  • Feature Engineering: Encoding, scaling, polynomial features, feature selection
  • Hyperparameter Tuning: GridSearchCV, cross-validation

from sklearn.pipeline import Pipeline from sklearn.preprocessing import StandardScaler from sklearn.ensemble import RandomForestClassifier from sklearn.model_selection import GridSearchCV pipeline = Pipeline([ ('scaler', StandardScaler()), ('clf', RandomForestClassifier(random_state=42)) ]) param_grid = { 'clf__n_estimators': [50, 100, 200], 'clf__max_depth': [None, 5, 10], } grid_search = GridSearchCV(pipeline, param_grid, cv=5, scoring='f1_weighted') grid_search.fit(X_train, y_train) print("Best params:", grid_search.best_params_)

Key Takeaways

  • Random Forest consistently outperformed single-tree models with proper hyperparameter tuning
  • Feature scaling had significant impact on distance-based models (KNN, SVM)
  • K-Means required careful initialization; DBSCAN was more robust to outliers
  • Cross-validation prevented overfitting and gave reliable generalization estimates
